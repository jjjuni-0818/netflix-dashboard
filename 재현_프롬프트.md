# Netflix 콘텐츠 대시보드 재현 프롬프트

---

Netflix 콘텐츠 데이터(8,859건)를 시각화하는 단일 HTML 대시보드를 만들어줘.
데이터는 Supabase REST API에서 async fetch로 불러오고, 외부 서버 없이 브라우저에서 직접 열 수 있어야 해.

## 디자인 토큰

- 메인 컬러: `#E50914` (Netflix Red)
- 배경: `#f5f5f0`, 카드: `#ffffff`
- 폰트: 헤더/KPI 숫자 → `Bebas Neue`, 본문 → `DM Sans` (Google Fonts CDN)
- border-radius: 카드 16px, 컴포넌트 10px

## 레이아웃 (위→아래 순서)

1. **헤더** — 로고("NETFLIX Content Database"), Supabase 배지, 우측에 총 콘텐츠 수·연도 범위·장르 수 3개 통계 표시
2. **로딩 오버레이** — 데이터 fetch 중 전체 화면을 덮는 스피너 + 진행 텍스트. fetch 완료 후 600ms 딜레이 뒤 사라짐
3. **에러 오버레이** — fetch 실패 시 전체 화면 표시, "다시 시도" 버튼(클릭 시 `location.reload()`)
4. **KPI 카드 3개** (가로 배열) — 페이지 로드 즉시 집계값으로 채워져야 함 (빈 '—' 상태 없이)
   - 총 콘텐츠 수 (현재 필터 기준)
   - 가장 많은 장르 + 편수
   - 연도 범위 (최소 – 최대)
5. **필터 바** — 장르 select, 등급 select, 연도 from–to 숫자 입력, 초기화 버튼. 변경 즉시 KPI·테이블 갱신
6. **차트 2개** (좌우 2열)
   - 좌: **장르별 가로 막대 차트** — 상위 15개 단일 장르 기준, Chart.js 4.x, 색상 `#E50914cc`, hover 시 `#E50914`, 클릭 시 해당 장르로 테이블 필터링
   - 우: **연도별 세로 막대 차트** — 전체 연도 범위, 클릭 시 해당 연도로 테이블 필터링
7. **콘텐츠 테이블**
   - 컬럼: 제목 / 연도 / 등급(배지) / 장르
   - 컬럼 헤더 클릭 시 오름차순↑·내림차순↓ 토글 정렬
   - 우측 상단 제목 검색 input (실시간 필터)
   - 차트 클릭으로 활성화된 필터는 태그(✕ 클릭으로 해제)로 표시
   - 페이지당 50건, 페이지네이션 (ellipsis 포함)
   - 기본 정렬: 연도 내림차순

## 데이터 스키마

Supabase `netflix_titles` 테이블에서 500건씩 배치 fetch:

```
{ name, year, rating, duration, category }
```

- `year`가 null이면 0으로 처리, 테이블에 '정보 없음' 표시
- `rating`이 null이면 'Unknown'
- `category`는 콤마+공백으로 구분된 복합 장르 문자열일 수 있음
  → 장르 차트·필터·KPI 계산 시 개별 장르로 분해(`split(', ')`)해서 집계

## 등급 배지 색상

- `All`, `TV-All` → 초록 계열
- `18+`, `NC-17` → 빨강 계열 (`rgba(229,9,20,0.1)`, 텍스트 `#E50914`)
- 나머지 → 기본

## 기술 스택

- Chart.js 4.4.1 (CDN)
- Supabase JS SDK (CDN) 또는 REST API 직접 호출
- 외부 프레임워크 없이 Vanilla JS
- 단일 HTML 파일
